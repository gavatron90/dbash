dbash

DBash

Tool for converting data between SQL and JS types easily. It's more like a test library to see how npm packages work.

Installation

Simply run npm install --save dbash or yarn add dbash

Usage

CommonJS:

var _ = require('dbash');
console.log(_.sqlIntJsBool(1));

ES2015:

import { sqlIntJsBool } from 'dbash';
console.log(sqlIntJsBool(1));

Also ES2015:

import * as _ from 'dbash';
console.log(_.sqlIntJsBool(1));

Booleans

jsBoolSqlInt(input: boolean | null): number | null Takes JavaScript boolean or null and returns 1, 0 or null

sqlIntJsBool(input: number | null): boolean | null Takes SQL number or null and returns true, false or null
